# 60059. 자물쇠와 열쇠

**Level 3**

https://programmers.co.kr/learn/courses/30/lessons/60059

# 풀이

완전탐색으로 키를 직접돌려가면서 모두 대조해본다. 

락을 기준으로 좌상단에 한칸만 보일때부터 우 하단에 한칸만 보일때까지 모든 경우의 수를 진행한다. 
key와 lock이 겹칠때에는 두개 배열의 합이 1이 되어야 하고, lock만 존재할때는 배열의 값이 1이여야 정답이 된다. 

그나마 대조하는 횟수를 줄이려고, 최소한의 key 구멍의 정사각형을 처음에 미리 찾아놓고 시작한다.
