# 풀이

간선들이 모두 주어져있고 이를 이용해서 경로를 찾는 문제. 
경로를 찾아주면 되는 문제인데, 경로가 두개 이상 있으면 사전순으로 줘야하므로, 탐색을 할 때 사전순으로 탐색을 하게 되면 무조건 첫번째껄 반환하면 된다. 
visited를 어떻게 만들어야할지 몰라서 map<vector<string>, bool>의 방식으로 만들었는데, 이는 나중에 테스트 케이스1이 계속 안되서 확인해보니까 중복 간선이 존재하는 경우라고 읽었다. 이를 고치려다가 다 고치진 못하고 지났다.
